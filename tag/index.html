    泛型

    # arguments 对象

    看起来像一个数组，可是并不是标准 Array 类型的实例，所以不能直接调用数组的方法。 但是可以通过 call 方法，从数组种提取。

    function callMethod(obj, method) {
  var args = [].slice.call(arguments, 2);
  return obj[method].apply(obj, args);
}

var obj = {
  add: function (x, y) {
    return x + y;
  },
};

callMethod(obj, "add", 1, 2); // 3
